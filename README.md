# Quiz Game 

This is a simple Quiz Game built using HTML, CSS and JavaScript.
The user answers multiple-choice questions within a limited time and gets a score based on correct answers.

## Features

* Multiple choice questions
* Timer for each question
* Score tracking (correct & wrong answers)
* Progress bar
* Start, Next and Restart functionality

## Technologies Used

* HTML
* CSS
* JavaScript
* Bootstrap

## How to Run

1. Download or clone this repository
2. Open the `quizgame.html` file in your browser
